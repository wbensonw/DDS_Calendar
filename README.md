# 澳門城市大學數據科學博士課程月曆

一個美觀、互動式、響應式的月曆網頁，專為澳門城市大學數據科學博士課程學生設計。此版本已更新至 2025-2026 學年第二學期。

## 📋 功能特色

### 🎨 設計特色
- **現代化設計**：採用 Apple 和 Google 的設計語言
- **糖果色彩**：使用柔和的糖果色調區分不同課程和活動
- **響應式佈局**：完美支援手機和電腦使用
- **優雅動畫**：流暢的過渡效果和互動反饋

### 📅 月曆功能
- **月份導航**：支援查看 2025 年 8 月至 2026 年 6 月的日程
- **多學期支援**：自動切換不同學期的課程表（2025秋、2026春）
- **課程顯示**：自動標記每週的固定課程
- **學校活動**：整合校曆中的重要日期和活動
- **今日高亮**：清楚顯示當前日期
- **詳情彈窗**：點擊日期查看當日課程和活動詳情

### 🎯 四象限佈局

日曆格子分為四個象限來顯示不同的工作班次：

```
┌─────────┬─────────┐
│  左上    │  右上    │
│ 晚上更   │ 下午更   │
│18:00-   │11:55-   │
│ 24:00   │18:25    │
├─────────┼─────────┤
│  左下    │  右下    │
│ 早上更   │ 通宵更   │
│06:25-   │23:35-   │
│ 12:25   │06:45    │
└─────────┴─────────┘
```

**顯示特色**：
- 象限之間的分割線為透明（不可見）
- 僅當日工作班次的象限會以藍色高亮顯示
- 休息日無象限高亮
- 時間衝突時象限會閃爍紅色

## 🚀 使用方法

1. **開啟網頁**：用瀏覽器打開 `index.html` 檔案
2. **導航月份**：使用左右箭頭按鈕切換月份
3. **查看詳情**：點擊任意日期查看當日課程和活動
4. **鍵盤操作**：使用 `Ctrl+←/→` 快速切換月份
5. **觸摸操作**：在手機上左右滑動切換月份

## ⚠️ 時間衝突檢測

系統會自動檢測工作時間與課程時間的衝突。例如，如果課程時間與工作時間重疊，日曆上會出現紅色警告標示。

### 衝突標示
- **🔴 脈衝動畫**：衝突日期的指示器會有紅色脈衝動畫
- **⚠️ 彈窗警告**：點擊衝突日期會顯示詳細的衝突警告
- **🎨 特殊樣式**：衝突的課程和工作都會有特殊的視覺標示

## 🔧 自訂功能

### 添加新課程
在 `script.js` 的 `courses` 物件中添加新課程：

```javascript
'NEW_COURSE': {
    name: '課程代碼',
    title: '課程名稱',
    day: 1, // 星期幾 (1=週一, 2=週二...)
    time: '上課時間',
    color: '顏色類別',
    location: '上課地點'
}
```

### 添加學校活動
在 `schoolEvents` 物件中添加新活動：

```javascript
'2026-MM-DD': {
    type: 'event', // 或 'holiday'
    title: '活動名稱',
    description: '活動描述',
    time: '活動時間'
}
```
